# Ignition-of-the-Atmosphere-With-Nuclear-Bombs-Rewrite
As soon as I saw the movie 'Oppenheimer' I had an interest to search for the original paper regarding the concern for the ignition of the atmosphere. 
I found the original declassified copy from the 40s but what do you know it was sort of hard to make out comfortably so I thought I'd sit down for a bit and just make it a little neater using an overleaf template.
Procrastination got the best of me but at last here it is in this repo. 

Note: I couldn't make out everything in the paper (hence some of the "?" in the rewrite) and there might also be some typos. 
If anyone catches anything or has some insight on what the "?" could be, please feel free to reach out to me on my LinkedIn which you can find on my README https://github.com/cbrianbrian/cbrianbrian
